✓ If you have not clicked any link:

✓ If you have clicked on a link:
  1. Change your password immediately if you have entered your user credentials into the phishing page.
  2. If the same password is used across many services (e.g. Gmail, Hotmail, Yahoo), those services passwords must be changed as well.
